6.1 Generalidades

Los codificadores y decodificadores permiten que el lenguaje de las máquinas (binario) sea **traducido e interpretado** por el usuario y otros dispositivos.

Aplicaciones de Enrutamiento:

  • Selección de direcciones de memoria.
  • Gestión de buses de datos compartidos.
  • Visualización numérica en displays de 7 segmentos.

6.5 Codificadores y Decodificadores

Codificador

Convierte un evento de entrada activa en un código binario compacto.

2^n ENTRADAS → n SALIDAS

Decodificador

Interpreta un código binario para activar una línea de salida específica.

n ENTRADAS → 2^n SALIDAS

"Un decodificador BCD a 7-Segmentos (74LS47) permite visualizar el número 15 usando dos displays controlados por el mismo bus."

6.6 Multiplexores y Demultiplexores

Multiplexor (MUX)

"El selector de señales"

Recibe varias entradas y encamina una hacia la salida según las líneas de control. Esencial para buses de datos compartidos.

Uso: Conmutadores de datos

Demultiplexor (DEMUX)

"El distribuidor de señales"

Toma una sola entrada y la dirige hacia una de las múltiples salidas disponibles. Proceso inverso al MUX.

6.7 Análisis de Comparadores

Un comparador de magnitud examina dos números (A y B) y determina si son **Mayores, Iguales o Menores**.

> A es Mayor que B
= A es Igual a B
< A es Menor que B
Ejemplo Industrial:

"En sistemas de control automático en Bolivia, se usa para comparar el nivel de un tanque con un setpoint y activar la bomba de llenado."
